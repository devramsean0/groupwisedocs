# The Mobile App

The mobile syncing relies on the "Immich" app that has to be downloaded from either the Google Play Store or the Apple App Store.

1. Download the app
2. Set the server url to be the url that you manage the web UI from
3. Enter the users email and password

## Choosing images to backup
1. Click the cloud button in the top corner
2. Press the "Select" button
3. Scroll down and press on/select individual albums or press "Select all"
4. Press the back chevron
5. Scroll down and press "Enable Backup" the switch should slide across

# The Desktop App
The website has the ability to be installed as a PWA. The steps for this depend on the browser, however for Microsoft edge it pops up on the menu bar with the wording "Install Immich app" which you can press. You cant back up through this though unfortunately
